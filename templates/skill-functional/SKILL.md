# Functional Skill Template

Use this skill when the task matches the declared functional capability and does not require project-private knowledge.

Do not use this skill for project-specific workflows unless a project skill explicitly delegates to it.

## Procedure

1. Confirm the task matches the triggers and not the negative triggers.
2. Load only the required project context.
3. Execute the capability.
4. Produce outputs in the declared output location.
5. Report assumptions and follow-up risks.
