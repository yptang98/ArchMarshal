# Common Project Skill Template

Use this skill for a reproducible engineering workflow that can be applied across projects.

Do not use this skill when required inputs, dependencies, or outputs are undeclared.

## Procedure

1. Read the project `AGENTS.md` and `.agent/INDEX.md`.
2. Check the skill manifest dependencies.
3. Run only local scripts from this skill directory.
4. Write reports or plans to declared output paths.
5. Do not apply file changes unless a future command has an explicit reviewed plan.
